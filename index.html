    /**
     * matcher.find()完成的任务：
     * 1. 根据指定的规则，定位满足规则的子字符串
     * 2. 找到子字符串后，将子字符串开始索引记录到matcher对象属性int[] groups中
     *    子字符串起始索引记录到groups[0]，子字符串的结束的索引+1记录到groups[1]
     * 3. 同时记录oldLast值为子字符串结束索引+1，即下次执行find时从oldLast开始
     *
     * matcher.group(0)：
     * 根据groups[0]和groups[1]的位置从content开始截取子字符串返回
     */

     * 非贪婪模式匹配搜索尽可能短的字符串，如在"ooooo"中，"o+?"只匹配单个"o"，而"o+"则匹配所有"o"

            System.out.println("找到目标结果：" + matcher.group(0));

    /**
     * 结巴程序
     */
    @Test
    public void test17(){
        String content = "11...2..3333..4....4";
        //1. 去掉所有的.
        Pattern pattern = Pattern.compile("\\.");
        Matcher matcher = pattern.matcher(content);
        content = matcher.replaceAll("");
        System.out.println(content);// 112333344
        //2. 去掉重复内容 (.)\\1+
        pattern = Pattern.compile("(.)\\1+");
        matcher = pattern.matcher(content);
        //3. 使用反向引用$1替换匹配到的内容
        content = matcher.replaceAll("$1");
        System.out.println(content);// 1234
    }
